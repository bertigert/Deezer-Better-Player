# Deezer-Better-Player
QoL improvements for the Deezer player(bar)

## Features:
Will probably add more in the future

- Allows you to toggle between remaining time and time played by clicking on the time played. <img width="614" height="76" alt="image" src="https://github.com/user-attachments/assets/ba6cde03-9ff9-4622-ac70-a5302a6c8352" />
- Changes the play previous track behaviour to only play the previous track if you have played a track for more than 5 seconds. Restarts the track otherwise.
